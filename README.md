# Ghazara | غزارة — Landing Page

AI-powered digital solutions for businesses and NGOs across the Middle East.

## Tech Stack
- React 18 + Vite
- Pure CSS-in-JS (no extra libraries)
- Bilingual: Arabic (RTL) + English (LTR)
- Mobile responsive

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
Output is in `/dist` — deploy to Netlify, Vercel, or GitHub Pages.

## Deploy on Netlify
1. Push to GitHub
2. Connect repo on [netlify.com](https://netlify.com)
3. Build command: `npm run build`
4. Publish directory: `dist`

## Deploy on Vercel
1. Connect repo on [vercel.com](https://vercel.com)
2. Framework: Vite
3. Auto-detected — just click Deploy
